## How to Declare an Array

1. `DATATYPE[] VARIABLE = {DATA1, DATA2, DATA3, ...};`

2. ```
   DATATYPE[] VARIABLE = new DATATYPE[length_of_array];
   DATATYPE[0] = value;
   DATATYPE[1] = value;
   ...
   ```
